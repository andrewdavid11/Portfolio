Styling Images

Styling the images on a website can really help them stand out, and can enhance and support the theme and feel of the entire site, creating a unique look that jumps from the millions of others on the Web.

Below, I am demonstrating many customized styles to apply to images. These type of effects can improve any site, but are of particular value for displaying images for sale, for example, bycreating a simulated frame or matte display.

This type of effect can be taken further with javascripts that allow the potential customer to cycle through different matte color combinations and frames to find their particular favorites. For a demo of that possibility, check out my "Card Color Picker" project.

On this page, the images are displayed without a table layout, so that the thumbnails readjust to any size of screen, in single column, or multiple. PHP and mySQL could be used to cycle through any number of images stored in a database, pulled up dynamically through searches and galleries for each visitor. Each image could start with the same frames, or have accents that match the particular image, as on this page.

As you can see, there are almost endless possibilities for styling and displaying images on modern websites. This is only a few of the possibilities to be achieved with relatively simple css.

A TV screen, reflective frame, wooden frame, colored mattes, and more effects can keep your site from becoming dull. Many more unique styles are out there. As an aspiring photographer, I always take the time to give images a customized look for any website.
